# RAVE-ONNX-Server
Server-side of the RAVE application - Licence Multimédia et Métiers du Web

# Installation
1 - Activate your virtual environment :
```bash
source path/to/miniconda3/bin/activate
conda activate my_virtual_env
```
2 - Clone this repository and install the dependencies :
```bash
git clone https://github.com/gnvIRCAM/RAVE-ONNX-Server.git
cd RAVE-ONNX-Server
pip install -r requirements.txt
```

3 - Run this command to start the python server : 
```bash
python server.py
```
